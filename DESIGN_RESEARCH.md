### Key Design Principles Summary

| Principle | Implementation |
|-----------|---------------|
| **Minimalism** | High whitespace, limited color palette, clean typography |
| **Nature-first** | Every section connects to the outdoor setting |
| **Food is hero** | Professional food photography front and center |
| **Storytelling** | Brand narrative woven throughout, not just an "About" page |
| **Mobile-first** | Design for phones first, enhance for desktop |
| **Performance** | Lazy loading, optimized images, minimal JS animations |
| **Accessibility** | Proper contrast, alt text, keyboard navigation, screen reader support |
| **Clear CTAs** | "Reserve" always visible, clear action hierarchy |
| **Social proof** | Reviews, Instagram feed, awards |
| **Urgency** | Live music schedule, seasonal menus, limited availability |
